Types of Functions

Function Types

  • Scalar Functions - Returns a single value. Used in SELECT, WHERE, etc.
  • Table-Valued Functions (TVF) - Returns a table. Can JOIN with other tables.
  • Inline TVF - Single SELECT statement returning table
  • Multi-statement TVF - BEGIN/END block returning table variable

Creating Scalar Functions

-- Scalar function to calculate bonus
CREATE FUNCTION dbo.CalculateBonus
    (@Salary DECIMAL(10,2), @Rating INT)
RETURNS DECIMAL(10,2)
AS
BEGIN
    DECLARE @BonusRate DECIMAL(3,2);
    
    SET @BonusRate = CASE
        WHEN @Rating >= 5 THEN 0.20
        WHEN @Rating >= 3 THEN 0.10
        ELSE 0.00
    END;
    
    RETURN @Salary * @BonusRate;
END

-- Use the function
SELECT first_name, salary, rating,
       dbo.CalculateBonus(salary, rating) AS bonus
FROM employees;

Table-Valued Functions

-- Inline table-valued function
CREATE FUNCTION dbo.GetEmployeesByDept
    (@DepartmentID INT)
RETURNS TABLE
AS
RETURN
(
    SELECT employee_id, first_name, last_name, salary
    FROM employees
    WHERE department_id = @DepartmentID
)

-- Use in JOIN
SELECT d.department_name, e.first_name, e.salary
FROM departments d
CROSS APPLY dbo.GetEmployeesByDept(d.department_id) e;

Performance Tip: Scalar functions can be slow on large datasets because they're called for each row. Consider using CROSS APPLY or rewriting with set-based operations.
